# go-gin-realworld-example-app

Example real world backend API built with Golang + Gin + Gorm

## 목표

- gin, gorm 리마인드
- Hexagonal Architectural 공부
- wire(DI), gomock 사용해보기
- Observability 적용해보기
- Github Action, ArgoCD 으로 CI/CD 적용해보기
